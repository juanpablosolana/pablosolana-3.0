La aplicación utiliza NextJS 12 para poder validar los datos ingresados, el framework permite crear endpoints al estilo de NodeJs por lo que ahora es muy simple y rápido consultar la información.

Para el aspecto grafico utilice Tailwind CSS lo que le da un aspecto moderno y responsivo.

El hosting fue gracias a Vercel ya que la integración con NextJS es espectacular (ya saben es el uno para el otro).

Al día de hoy solo está disponible la consulta llenando el formulario, pero en pocos días voy a liberar la opción de subir el CFDI.

Este proyecto quiere ayudar a todos los ciudadanos a verificar si las facturas que les dan o emiten está legalmente procesadas por el SAT MX y conocer es estatus ya que se pueden cancelar.
El api va a estar disponible para todos, pero estoy agregando JWT para proteger el servidor de posibles abusos.

Saludos y no olvides verificar el estatus de tus comprobantes fiscales digitales por internet.
